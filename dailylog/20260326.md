## 討論與決策（Q&A）

Q: 公司電腦 Docker port 要用哪個？
A: 改用 5433，因為本機已有 PostgreSQL 佔用 5432

Q: 刪除功能要開放嗎？
A: 不開放硬刪除，改用停用/啟用取代，ERP 標準做法

Q: 登入狀態要用什麼機制保持？
A: JWT + localStorage，token 有效期 8h（一個工作天）
   - token 存在且未過期 → 維持登入
   - token 過期或不存在 → 導回登入頁
   - 主動登出 → 清除 localStorage → 導回登入頁

Q: role-view 全選要怎麼設計？
A: 橫向全選（每列一個、每個 Module 一個），
   上方保留「全部勾選」「全部取消」快捷按鈕

Q: 首頁設計方向？
A: 上下分區，顯示：快捷入口、簡易數據概覽、
   待辦事項（左）、系統公告（右）、登入者資訊
