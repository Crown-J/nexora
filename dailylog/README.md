# NEXORA Daily Log

每日工作記錄，供 Alex（PM AI）與 Hank（Cursor）跨對話銜接用。

撰寫規範請見專案根目錄 [.cursorrules](../.cursorrules) 中的 **「Daily Log（工作日誌）」** 一節。

## 命名規則
- 格式：`YYYYMMDD.md`
- 例如：`20260326.md`

## 每日記錄格式
- 以「當日日期」為檔名的檔案內，依範本區塊填寫（專案資訊、今日完成、Q&A、明日待辦、問題與解法）。
- **內容須為當日真實工作**；任務指令裡若附帶範例文字，僅供格式參考，**不可**原樣當成已完成事項寫入。
